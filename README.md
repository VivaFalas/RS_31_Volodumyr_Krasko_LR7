# RS_31_Volodumyr_Krasko_LR7
Номер групи РС-31 Автор: Красько В. A. ОПЕРАЦІЙНІ СИСТЕМИ РЕАЛЬНОГО ЧАСУ ТА 32 БІТНІ МК

Лабораторна робота 7 Налагодження середовища розробки для програмування
32-розрядних мікроконтролерів з використанням операційної системи реального часу. На
прикладі FreeRTOS. 

Обладнання: STM32F411 , CubeMX, Keil.

Мета: Ознайомитися з принципами роботи операційної системи реального часу
FreeRTOS, навчитися створювати та керувати потоками для керування світлодіодами на
STM32F411 Discovery.

Варіант 9


  1. Створити три потоки для керування світлодіодами PD12, PD13, PD15. Частоти: PD12 — 1 Гц 
(тривалість 200 мс), PD13 — 2 Гц (тривалість 100 мс), PD15 — 5 Гц (тривалість 50 мс). Кнопка 
PA0 змінює тривалість імпульсу PD12 від 200 мс до 500 мс.
